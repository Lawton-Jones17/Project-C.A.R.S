------------------------- PROJECT DESCRIPTION ------------------------------

This project is a wireless control system for the Lafvin Mecanum-wheeled car 
implementing wireless communication protocols using ESP32 microcontrollers. 
We will use a custom steering wheel and Logitech foot pedals as HID (Human 
Interface Device) inputs to control the rover. This project will have a focus 
on wireless data transmission, interfacing and decoding signals from HID 
inputs, and precise motor actuation.


------------------------------- HARDWARE -----------------------------------

- Arduino UNO
- ESP32DevKit (3)
- Mecanum Wheel Car Kit
- MPU6500 Accelerometer Sensor for Steering
- Logitech Pedals with DB9 Connector (Dual Potenitometer Input)
- Wheel and Pedal Wire Housing
